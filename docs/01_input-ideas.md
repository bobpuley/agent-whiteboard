**FR27 — Dark Theme & Theme Selector (2026-07-18)**

- Feature request: add a dark theme and a theme selector for the board UI. Theme applies only to the board chrome (toolbar, panels, controls) — explicitly **not** to rendered content (the exported/embedded HTML shown on the canvas). Selection persists across sessions (localStorage); default for first-time users is Light. Selector is a simple toggle in the board toolbar.
