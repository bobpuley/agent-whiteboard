History button — icon style options

The history button currently uses a raw emoji (🕑), while delete and export use clean line-style SVG icons (24×24 viewBox, stroke="currentColor", stroke-width="2.2", round caps/joins — a Feather-icon-style outline). Each card below is the real .controls-panel markup/CSS from App.svelte, with only the history icon swapped. Hover any button to see its real hover state.

Current (emoji)
baseline — not in the SVG line-icon style

closest literal match to the current emoji's meaning
B — Rotate-back arrow
common "history/revert" convention (undo-style)
C — List
emphasizes "a list of past snapshots"
D — Archive box
"a box of saved past snapshots"
E — Layers / stack
"multiple past states stacked up"
Each history button above also gets an accent hover state (light blue, matching export's hover) instead of the current plain-grey hover, purely so it reads as "part of the same button family" — easy to drop if you'd rather keep the neutral hover it has today. Source icons: Feather Icons outline set (same visual language as the existing delete/export glyphs).
